fizzBuzz

  1. Set the Range: Enter a start and an end number.
  2. Configure FizzBuzz: Enter your Fizz and Buzz numbers. Optionally, add custom labels for Fizz, Buzz, and FizzBuzz.
  3. Play the Game: Click Start to begin counting. Numbers will update every second: Shows FizzBuzz if divisible by both, Fizz if divisible by the Fizz number, Buzz if divisible by the Buzz number, Otherwise, the number itself.
  4. Control: Pause/Resume the game anytime. Click Stop to end the game. Use Reset to clear and start over.
